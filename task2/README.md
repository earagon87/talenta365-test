# Task 2: Programming Challenge

Application in Python that receives the number of excel column and returns what is the conversion into letters. In the same way Excel does, BUT including "Ñ" letter.

Test Cases

- 1 -> A
- 2 -> B
- 13 -> M
- 14 -> N
- 15 -> Ñ
- 16 -> O
- 28 -> AA
- 54 -> AZ
- 55 -> BA
- 69 -> BÑ
