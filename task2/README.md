# Task 2: Programming Challenge

Application in Python that receives the number of a excel column and say what is the conversion into letters. In the same way Excel does, BUT adding the letter "Ñ".

Test Cases

- 1 -> A
- 2 -> B
- 13 -> M
- 14 -> N
- 15 -> Ñ
- 16 -> O
- 28 -> AA
- 54 -> AZ
- 55 -> BA
- 69 -> BÑ
